# PhoenixPME Protocol

[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](LICENSE)

A decentralized protocol for peer-to-peer trading of physical precious metals using bonded auctions and on-chain reputation.

## Features
- **Tokenized Real-World Assets**: Gold, silver, platinum, palladium
- **Reserve-Price Auctions**: Sellers set minimum prices
- **Multi-Asset Settlement**: Bid with XRP, COREUM, SOLO, wBTC, or any IBC-compatible asset
- **Automatic Penalty System**: Bond forfeiture for failed shipments
- **Immutable Reputation**: On-chain feedback that can't be censored
